# Smart-Q – Smart Queue Management System

A full-stack web application for managing queues and events efficiently.

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB Atlas

## Features
- Queue token generation
- Admin queue control
- Event scheduling
- REST API based architecture

## Project Structure
Smart-Q/
├── frontend/
├── backend/

## How to Run

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm start
